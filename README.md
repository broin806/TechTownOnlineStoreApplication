
A full-stack Node.js + Express + MySQL web application that processes customer orders, stores them in a database, and create receipts.

## Used these technologies: 
- Node.js
- Express.js
- MySQL
- EJS

##  Features:
- Form validation for customer input
- Session-based receipt storage
- MySQL database integration

## Project Purpose: 
This project demonstrates backend development skills including:

- REST-style routing with Express  
- Server-side validation    
- SQL database integration
